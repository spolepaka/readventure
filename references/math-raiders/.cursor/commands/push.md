# Push

```
/push
```

Automatically:
1. Analyzes your changes with `git diff`
2. Generates commit message using AI
3. Runs `git add -A && git commit -m "message" && git push`

## Examples
```
/push              # AI generates: "feat: add boss health bar animation"
/push              # AI generates: "fix: resolve null pointer in combat system"
/push              # AI generates: "refactor: extract damage calculation to utils"
```

Zero typing. Just `/push`.
